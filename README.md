### 安装步骤

复制到本地，将chess目录用终端打开，输入npm install，下载成功后再输入npm start即可运行。

### 功能

简单的井字棋玩法，可提示当前回合轮到谁和状态。

### 不足

提示功能（Notice组件）存在问题，不知道如何实时传值、渲染状态

### 预计改进
