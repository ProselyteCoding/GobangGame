### 安装步骤

git clone到本地，将chess目录用终端打开，输入npm install，下载成功后再输入npm start即可运行。

### 功能

简单的五子棋玩法，可提示当前回合轮到谁和状态。

### 不足

功能比较简单，未实现后端或者数据库

### 预计改进

· 实现记忆功能   <br/>
· 在此基础上设计其他简单棋类游戏如五子棋、围棋、国际象棋等
